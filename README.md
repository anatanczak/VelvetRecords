# VelvetRecords
Mini project to learn MVC, CRUD, PDO using PHP. 

Un petit projet pour maîtriser MVC, CRUD, PDO avec PHP. 

**Les fonctionnalités:**

- Connexion/Déconnexion 
- Affichage d'une liste de produits
- Ajout d'un produit
- Suppression d'un produit
- Modification d'un produit
